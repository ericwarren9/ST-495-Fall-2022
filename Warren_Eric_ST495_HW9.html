From our tests we know the null hypothesis is that there is no association between variables and from the alternative that there is association. The \(\alpha\) level for these tests are 0.05.

  • We can see from the Hair and Eye variables that the p-value from this test is 2.3252868^{-25}. From this result, we can reject the null hypothesis and there is an association between hair and eye colors.

  • We can see from the Hair and Sex variables that the p-value from this test is 0.0461308. From this result, we can reject the null hypothesis and there is an association between hair color and a person’s sex.

  • We can see from the Sex and Eye variables that the p-value from this test is 0.6754042. From this result, we can fail to reject the null hypothesis and conclude that there is most likely NOT an association between a person’s sex and eye color.

  • Lastly, we can see from the Cochran-Mantel-Haenszel test that there was some kind of association with our variables when they were all together with the p-value of 9.016374^{-26}. We can see from before that the hair color is associated with the other two variables which causes this result.
